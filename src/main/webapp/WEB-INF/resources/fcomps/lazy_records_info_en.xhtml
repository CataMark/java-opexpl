Sorting records by multiple columns:

  • hold down Ctrl key and click on multiple columns
  • for reverting to sorting on a single column, click on the desired one without holding down Ctrl key

Table filtering:

  • for applying the filters hit Enter key

Symbols used for table filtering:

Symbol Description Example % Percentage character:
  • finds any text with any length including zero length text
  • %auto% - finds any text that contains "auto"
  • auto% - finds any text that starts with "auto"
  • %bus - finds any text that ends with "auto"
 _ Underscore character:
  • replaces one character
  • _ontrolling - finds any text that has a length of 11 characters and ends with "ontrolling"
 [ ] Characters between square brackets:
  • finds any character that matched the interval or list of specified characters
  • [a-m]test - finds any text with a length of 5, starts with a character included in the "a-m" interval and ends with "test"
  • [abc]test - finds any text with a length of 5, starts with a character from "abc" list and ends with "test"
  • [a-cms]test - combo between interval and list
 [^] Exclusion characters:
  • finds any character that is not included in the specific interval or list
  • [^a-m]test - finds any text with a length of 5, does not start with a character included in the "a-m" interval and ends with "test"
  • [^abc]test - finds any text with a length of 5, does not start with a character from "abc" list and ends with "test"
  • [^a-cms]test - combo between interval and list
 []% Null values, does not exclude texts with zero length (in Excel the zero length text is represented by a pair of double quotes with nothing in between: "").
  • this character sequence is specific to this app (added programmatic)
 [^]% Non null values.
  • all texts, including zero length text
 Combos:
  • de[^a]% - finds any text that starts with "de", the following character is not "a" and end with any number of characters (or none)

Filtering date fields:

  • the format for filtering this type of field is YYYY-MM-DD HH:mm:ss (eg.: 2019-01-05 13:12:10, 2019-01-05%, %-01-05 13:12:%).
